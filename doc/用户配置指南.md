## 快速开始

OneHand 安装后，**语音识别功能开箱即用**，无需额外配置。但如果您需要使用 AI 智能回答功能，则需要配置 AI 服务。

---

## AI 大模型配置

AI 功能用于在语音转文字后，自动生成智能回答。您需要配置一个兼容 OpenAI API 格式的服务。

### 步骤 1：进入设置页面

1. 启动 OneHand 应用
2. 点击左上角的设置图标（齿轮图标），或在主页找到「设置」入口
3. 进入设置页面

### 步骤 2：配置 API 信息

在「大模型配置」区域，填写以下信息：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **服务提供商** | 选择 API 提供商类型 | OpenAI兼容API |
| **API Key** | 您的 API 密钥 | `sk-xxxxxxxxxxxx` |
| **Base URL** | API 服务地址 | `https://api.openai.com/v1` |
| **模型** | 使用的模型名称 | `gpt-4o` 或 `qwen-turbo` |

### 步骤 3：保存配置

修改设置后，系统会自动保存，无需手动操作。

---

### 推荐的 AI 服务商

以下是几个常用的兼容 OpenAI API 格式的服务商：

#### 1. ModelScope（魔搭社区）- 默认配置

- **Base URL**: `https://api-inference.modelscope.cn/v1`
- **模型**: `Qwen/Qwen3-235B-A22B-Instruct-2507`、`Qwen/Qwen3-VL-235B-A22B-Instruct`
- **获取方式**: 访问 [ModelScope](https://modelscope.cn/) 注册账号并获取 API Key

#### 2. OpenAI

- **Base URL**: `https://api.openai.com/v1`
- **模型**: `gpt-4o`、`gpt-4o-mini` 等
- **获取方式**: 访问 [OpenAI Platform](https://platform.openai.com/) 注册并获取 API Key

#### 3. 阿里云百炼

- **Base URL**: `https://dashscope.aliyuncs.com/compatible-mode/v1`
- **模型**: `qwen-turbo`、`qwen-plus`、`qwen-max` 等
- **获取方式**: 访问 [阿里云百炼](https://bailian.console.aliyun.com/) 开通服务并获取 API Key

#### 4. DeepSeek

- **Base URL**: `https://api.deepseek.com/v1`
- **模型**: `deepseek-chat`、`deepseek-coder` 等
- **获取方式**: 访问 [DeepSeek](https://platform.deepseek.com/) 注册并获取 API Key

#### 5. 其他兼容服务

任何兼容 OpenAI API 格式的服务都可以使用，例如：
- 本地部署的 Ollama
- 其他云服务商（如智谱 AI、硅基流动等）

只需将 **Base URL** 改为服务商提供的地址，**模型** 改为支持的模型名称即可。

---

### 配置示例

**使用 ModelScope 服务：**

```
API Key:   您的 ModelScope API Key
Base URL:  https://api-inference.modelscope.cn/v1
模型:      Qwen/Qwen3-235B-A22B-Instruct-2507
```

**使用 OpenAI 服务：**

```
API Key:   您的 OpenAI API Key
Base URL:  https://api.openai.com/v1
模型:      gpt-4o
```

**使用阿里云百炼服务：**

```
API Key:   您的阿里云 API Key
Base URL:  https://dashscope.aliyuncs.com/compatible-mode/v1
模型:      qwen-turbo
```

## 界面设置

在设置页面的「其他设置」区域，您可以配置：

### 界面语言

- **跟随系统**：自动使用操作系统的语言设置
- **中文**：强制使用中文界面
- **English**：强制使用英文界面

### 主题

- **跟随系统**：自动使用操作系统的深色/浅色模式
- **浅色**：强制使用浅色主题
- **深色**：强制使用深色主题

## 常见问题

### Q: 语音识别需要联网吗？

**A: 不需要。** OneHand 使用本地离线语音识别引擎（Sherpa-ONNX），所有语音转文字都在您的电脑本地完成，无需联网，保护您的隐私。

### Q: AI 回答功能需要联网吗？

**A: 需要。** AI 智能回答功能需要连接您配置的 AI 服务。如果您配置的是云端服务（如 OpenAI、ModelScope），则需要联网。如果配置的是本地部署的服务（如 Ollama），则不需要联网。

### Q: API Key 填错了怎么办？

**A: 直接在设置页面修改并保存即可。** 如果 AI 回答功能无法使用，请检查：
1. API Key 是否正确
2. Base URL 是否正确（注意末尾不要有多余的斜杠）
3. 模型名称是否正确
4. 网络是否能访问对应的服务

### Q: 配置文件保存在哪里？

**A: 配置文件保存在系统用户数据目录：**
- **Windows**: `%APPDATA%/OneHand/config.json`
- **macOS**: `~/Library/Application Support/OneHand/config.json`

如需备份，可以复制该文件。

### Q: 如何验证 AI 配置是否正确？

**A:** 在画布中创建一条语音或文本笔记，系统会自动尝试生成 AI 回答。如果配置正确，您会看到 AI 生成的回答内容；如果配置有误，会显示错误提示。

### Q: 支持哪些语言进行语音识别？

**A: 当前版本默认使用中文语音识别模型，适合识别普通话。建议在安静环境下使用标准普通话进行录音，以获得最佳识别效果。
