# Sherpa-Onnx 集成检查清单

## 📋 集成前检查

### 1. 环境准备
- [ ] Node.js 版本 >= 16
- [ ] npm 或 yarn 已安装

### 2. 依赖安装
```bash
npm install
```
- [ ] 依赖安装成功
- [ ] 无错误信息

### 1. 下载模型
```bash
npm run download:sherpa-model
```

### 2. 验证下载
- [ ] 控制台显示 "下载完成!"
- [ ] 控制台显示 "解压完成!"
- [ ] 控制台显示 "✓ 模型文件验证成功"

### 3. 检查文件

应该看到：
- [ ] `model.int8.onnx` (约 86MB)
- [ ] `tokens.txt` (几 KB)
